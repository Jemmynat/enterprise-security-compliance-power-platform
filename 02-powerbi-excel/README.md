# Security Compliance Analytics Layer  
*(Power BI & Excel)*

## Purpose
This module represents the analytics layer of the enterprise security platform.  
It was designed to provide centralized, real-time visibility into endpoint security compliance across regions, states, departments, and security solutions.

---

## Business Problems Addressed
- Fragmented reports from multiple security tools  
- Lack of executive visibility into compliance posture  
- No geographic or departmental compliance comparison  
- Manual compliance tracking  

---

## Key Features

### ✔ Enterprise Compliance Overview
- Total assets monitored  
- Overall compliance score  
- Fully non-compliant systems  
- Total states, regions, and departments  

### ✔ Security Solution Compliance
- Antivirus compliance  
- Disk encryption compliance  
- Asset inventory agent compliance  
- Patch management compliance  
- Data loss prevention compliance  

### ✔ Geographic Compliance
- Interactive Nigeria compliance map  
- Regional compliance comparisons  
- State-level performance analysis  

### ✔ Performance Ranking
- Top and bottom states  
- Top regions  
- Top departments  
- Top IT support teams  

### ✔ Asset-Level Visibility
- Hostname-level compliance  
- Assigned IT support and managers  
- Per-solution compliance tracking
